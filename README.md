# Market-Basket Analysis on IMDB Movies Dataset

## Project Description
This project applies market-basket analysis to the IMDB Movies Dataset. Movies are treated as baskets, and actors are treated as items.

## Dataset
The dataset is downloaded from Kaggle using the Kaggle API.

## Method
The A-Priori algorithm is used to find frequent actor combinations. Association rules are then generated using a minimum confidence threshold.

## Results
The analysis found frequent actor combinations, especially involving Daniel Radcliffe, Emma Watson, and Rupert Grint.
